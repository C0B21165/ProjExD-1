## packman（ex06/packman.py）
### ゲーム概要
- ex06/packman.pyを実行すると，440x280のスクリーンに迷路のようなものが描画され，
　赤玉を移動させ敵をよけてゴールするゲーム
- 赤玉が敵と接触するとゲームオーバーで終了する
### 操作方法
- 矢印キーで赤玉を上下左右に移動する
